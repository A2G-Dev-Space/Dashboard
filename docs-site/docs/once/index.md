## 핵심 컨셉

::: warning 중요 — 사용자가 직접 편집하지 않습니다
ONCE는 **모든 노트 작성/수정이 AI를 통해서만** 이루어집니다.
원하는 내용을 자연어로 입력하면 AI가 알아서 정리합니다.
:::

### 이런 상황에서 사용하세요

| 상황 | 입력 예시 |
|------|----------|
| 회의록 정리 | "오늘 회의 참석자는 김팀장, 이과장이고 논의 내용은..." |
| 아이디어 정리 | "새로운 프로젝트 아이디어가 있는데, 핵심은..." |
| 팀 지식 공유 | "팀원들에게 알려줄 내용인데, API 사용법은..." |
| 내용 검색 | "지난주 마케팅 회의에서 결정된 예산 관련 내용 찾아줘" |
