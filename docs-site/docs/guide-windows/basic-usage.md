# Basic Usage

## 인터페이스 소개

Nexus Coder for Windows는 **채팅형 GUI**를 제공합니다.

| 영역 | 설명 |
|------|------|
| **사이드바** | 세션 목록, 설정, 도구 선택 |
| **채팅 영역** | AI와의 대화 내용 (마크다운 렌더링) |
| **입력창** | 자연어로 요청 입력 |
| **상태바** | 모델 정보, 토큰 사용량, 연결 상태 |

## 기본 작업

채팅 입력창에 자연어로 요청을 입력합니다:

### 코드 수정
```
login 함수의 버그를 수정해줘
이 함수에 에러 처리를 추가해줘
```

### 파일 탐색
```
src 폴더의 구조를 알려줘
package.json에서 버전을 확인해줘
```

### 프로젝트 분석
```
이 프로젝트에서 인증 관련 코드가 어디에 있어?
성능 개선점이 있는지 분석해줘
```

### 코드 생성
```
새로운 API 엔드포인트를 만들어줘
UserService 클래스를 생성해줘
```

## 작업 디렉토리 설정

사이드바 상단에서 **작업 디렉토리**를 설정합니다. AI는 해당 디렉토리의 파일을 읽고 수정할 수 있습니다.

::: tip
프로젝트 루트 디렉토리를 작업 디렉토리로 설정하면 전체 프로젝트를 분석하고 수정할 수 있습니다.
:::

## 도구 활성화

사이드바의 **Tool Selector**에서 추가 도구를 활성화할 수 있습니다:

| 도구 | 설명 |
|------|------|
| **Browser Tools** | Chrome/Edge 자동화 (웹 테스트, 스크린샷) |
| **Office Tools** | Word, Excel, PowerPoint 자동 조작 |

::: warning 도구 사용 시 주의
- Browser Tools: Chrome 또는 Edge가 설치되어 있어야 합니다
- Office Tools: Microsoft Office가 설치되어 있어야 합니다
:::

## 세션 관리

### 새 세션
사이드바에서 **+ New Session** 버튼으로 새 대화를 시작합니다.

### 세션 전환
사이드바의 세션 목록에서 이전 대화를 클릭하여 전환합니다.

### 세션 삭제
세션 항목에서 삭제 버튼을 클릭합니다.

## 모델 전환

상태바의 **모델 이름**을 클릭하여 사용 가능한 LLM 모델 목록을 확인하고 전환할 수 있습니다.

::: info
사용 가능한 모델은 관리자가 A2G 서버에 등록한 모델입니다. 사업부별 허용 모델이 다를 수 있습니다.
:::

## 설정

사이드바의 **Settings** 메뉴에서 다음을 설정할 수 있습니다:

- **작업 디렉토리** - 기본 작업 경로
- **테마** - 다크/라이트 모드
- **로그 레벨** - 디버그 로그 활성화
- **SSO 정보** - 현재 로그인 상태 확인
