# FAQ

## 일반

### 접속이 안 돼요
사내 네트워크(VPN 포함)에 연결되어 있는지 확인하세요. FREE는 사내망에서만 접속 가능합니다.

접속 URL: **http://a2g.samsungds.net:15001**

### 로그인이 안 돼요
Samsung SSO 인증을 사용합니다. SSO 서버에 문제가 있을 수 있으니, 잠시 후 다시 시도하세요. 세션은 24시간 유지됩니다.

### 팀이 자동으로 매칭되지 않아요
SSO 로그인 시 부서명(`deptname`)에서 팀명을 추출합니다. 예: `AI플랫폼팀(DS부문)` → 팀명 `AI플랫폼팀`. 부서명 형식이 다른 경우 syngha.han에게 문의하세요.

## 업무 입력

### 입력이 안 돼요 / AI가 응답하지 않아요
LLM 프록시 서버 상태를 확인해 주세요. SuperAdmin에게 LLM 설정 상태를 문의하세요.

### 입력한 내용이 이상하게 분리되었어요
AI가 업무 항목을 잘못 분리한 경우, 각 항목의 제목과 내용을 직접 수정할 수 있습니다. 항목을 클릭하면 편집 모드로 전환됩니다.

### 과거 날짜로 입력할 수 있나요?
네, 최대 29일 전까지 날짜를 지정할 수 있습니다. 입력 시 날짜 선택기에서 원하는 날짜를 선택하세요.

## 보고서

### 보고서가 생성되지 않았어요
자동 보고서는 매주 월요일에 생성됩니다. 해당 기간에 입력된 업무 항목이 없으면 보고서가 생성되지 않을 수 있습니다.

### 보고서를 수동으로 생성할 수 있나요?
SuperAdmin 권한이 필요합니다. 관리자 페이지에서 수동 생성이 가능합니다.

### 보고서가 사라졌어요
보고서는 30일간 보관됩니다. 30일이 지나면 자동으로 삭제됩니다.

## 조직

### 그룹/파트를 변경하고 싶어요
프로필 페이지에서 소속 그룹/파트를 변경할 수 있습니다. 변경 시 이전 그룹/파트에서 사용자가 빠지며, 비어있는 그룹/파트는 자동 삭제됩니다.

### 그룹/파트 이름 규칙이 있나요?
긴 영문명은 축약을 권장합니다:
- `Agent Enabler` → `AE그룹` / `AE파트`
- `Platform Development` → `PD파트`

입력 시 AI가 자동으로 정규화해 줍니다.
