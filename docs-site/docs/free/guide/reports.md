# 보고서

## 자동 주간 보고서

FREE는 매주 월요일에 자동으로 주간 보고서를 생성합니다.

### 생성 순서

```
1. 파트 보고서 (Part)    ← 파트원 개인 업무 취합
2. 그룹 보고서 (Group)   ← 파트 보고서 취합
3. 팀 보고서 (Team)      ← 그룹 보고서 취합
```

### 보고서 뷰

각 보고서는 **두 가지 뷰**를 제공합니다:

| 뷰 | 설명 |
|----|------|
| **개인별 정리** | 각 구성원이 수행한 업무를 인원별로 정리 |
| **항목별 정리** | 유사한 업무를 항목별로 통합하고, 참여 인원 명시 |

### 보고서 확인

1. 좌측 사이드바에서 원하는 공간(파트/그룹/팀)으로 이동
2. **보고서** 탭을 클릭
3. 보고서 목록에서 원하는 날짜의 보고서 선택

### 보고서 유효기간

보고서는 **30일간** 보관됩니다. 30일이 지나면 자동으로 삭제됩니다.

## 내보내기

보고서를 다양한 형식으로 다운로드할 수 있습니다:

| 형식 | 확장자 | 용도 |
|------|--------|------|
| **Word** | `.docx` | 공식 보고서 제출용 |
| **Excel** | `.xlsx` | 데이터 분석, 가공용 |
| **Markdown** | `.md` | 위키, 문서 시스템 연동용 |

### 다운로드 방법

1. 보고서 상세 페이지에서 **다운로드** 버튼 클릭
2. 원하는 형식 선택
3. 파일명은 자동 생성됩니다 (예: `AI플랫폼팀_주간보고서_2026-01-27.docx`)

::: tip 파트/그룹/팀 공간에서도 다운로드
각 공간의 보고서 영역에서 바로 다운로드할 수 있습니다.
:::

## 수동 보고서 생성

관리자(SuperAdmin)는 스케줄을 기다리지 않고 수동으로 보고서를 생성할 수 있습니다.

::: info
자세한 내용은 [관리자 기능](/free/guide/admin)을 참고하세요.
:::
