# 관리자 기능

## 역할 구조

FREE는 3단계 권한 구조를 가집니다:

| 역할 | 권한 | 설정 방법 |
|------|------|----------|
| **일반 사용자** | 개인 업무 입력, 소속 공간 보고서 조회 | 기본 |
| **팀 관리자** | 해당 팀 내 그룹/파트/구성원 관리 | SuperAdmin이 지정 |
| **SuperAdmin** | 전체 시스템 관리, LLM 설정, 모든 팀 관리 | 환경변수 `DEVELOPERS` |

## SuperAdmin 기능

### LLM 관리

- **모델 동기화**: LLM 프록시에서 사용 가능한 모델 목록을 동기화
- **활성 모델 변경**: AI 응답에 사용할 모델 선택
- **Endpoint 설정**: LLM API 엔드포인트 변경

### 팀 관리

- 모든 팀의 구성원 조회
- 팀 관리자 지정/해제
- 활동 로그 조회

### 시스템 관리

- 공지사항 관리 (팀별)
- 보고서 생성 현황 모니터링

## 팀 관리자 기능

### 구성원 관리

- 팀 내 구성원 목록 조회
- 그룹/파트 배정 변경

### 공지사항

- 팀 공지사항 작성/수정
- 로그인 시 팀원에게 표시됩니다

::: warning 팀 범위 제한
팀 관리자는 자신이 관리하는 팀의 데이터만 접근할 수 있습니다.
다른 팀의 데이터는 볼 수 없습니다.
:::
