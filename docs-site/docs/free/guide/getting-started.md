# 시작하기

주간 보고 취합에서 해방되세요, 알아서 취합해 드릴게요. FREE는 AI powered 주간보고 취합 서비스입니다.
업무 내용을 자유롭게 입력하면 AI가 자동으로 정리하고, 매일 자정 지난 7일간의 보고서를 취합합니다.

## 1단계: 접속

브라우저에서 다음 주소로 접속합니다:

::: tip 접속 URL
**[http://a2g.samsungds.net:15001](http://a2g.samsungds.net:15001)**
:::

::: warning 사내망 필수
사내 네트워크에 연결된 상태에서만 접속할 수 있습니다.
:::

## 2단계: SSO 로그인

- SSO 로그인 페이지로 자동 리다이렉트됩니다
- 사내 계정(Samsung SSO)으로 로그인하세요
- 세션은 24시간 유지됩니다 (만료 시 재로그인)

## 3단계: 조직 설정 (온보딩)

최초 로그인 시 소속 **그룹**과 **파트**를 설정합니다.

1. **그룹 선택**: 기존 그룹에서 선택하거나 새로 생성
2. **파트 선택**: 기존 파트에서 선택하거나 새로 생성

::: info 이름 정규화
입력한 이름은 AI가 자동으로 정규화합니다.
- `Agent Enabler` → `AE그룹` / `AE파트`
- `platform team` → `Platform팀`

긴 영문명은 축약하여 입력하는 것을 권장합니다.
:::

::: tip 팀 자동 매칭
SSO 로그인 정보에서 팀명이 자동으로 추출됩니다.
별도 설정 없이 같은 팀 소속 사원은 동일한 팀 공간을 공유합니다.
:::

## 4단계: 첫 업무 입력

1. **개인 공간**에서 입력창에 업무 내용을 입력합니다
2. 날짜를 선택합니다 (기본: 오늘)
3. **전송** 버튼을 누릅니다
4. AI가 내용을 분석하고 개별 업무 항목으로 분리합니다

**입력 예시:**
```
오늘 킥오프 미팅 진행했고, API 스펙 확정했습니다.
오후에는 PR #42 코드리뷰 완료하고 피드백 반영했습니다.
내일 QA 테스트 예정입니다.
```

AI가 이 입력을 3개의 개별 업무 항목으로 자동 분리합니다.

::: tip
형식에 얽매이지 않아도 됩니다. Jira 이슈, 메신저 대화, 회의록 등 아무거나 붙여넣어도 AI가 업무 항목을 추출합니다.
:::
