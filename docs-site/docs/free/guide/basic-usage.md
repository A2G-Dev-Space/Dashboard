# 기본 사용법

## 업무 입력

### 개인 공간에서 입력

1. 좌측 사이드바에서 **개인 공간**을 클릭합니다
2. 상단 입력창에 업무 내용을 자유롭게 입력합니다
3. 날짜를 선택합니다 (기본: 오늘, 최대 29일 전까지)
4. **전송** 버튼을 클릭합니다

::: tip 다양한 입력 형태 지원
- Jira 이슈 내용 복사/붙여넣기
- 메신저/채팅 이력 붙여넣기
- 이메일 본문 복사
- 회의록, 메모
- 직접 작성한 자유 텍스트
:::

### AI 자동 분리

하나의 입력에서 여러 업무 항목이 자동으로 분리됩니다.

**입력:**
```
오전에 코드리뷰 3건 완료, 오후에 DB 마이그레이션 스크립트 작성.
저녁에 장애 대응 - Redis 메모리 이슈 해결함.
```

**결과:** 3개 항목으로 분리
| 제목 | 내용 | 날짜 |
|------|------|------|
| 코드리뷰 완료 | 코드리뷰 3건 완료 | 2026-01-30 |
| DB 마이그레이션 스크립트 작성 | DB 마이그레이션 스크립트 작성 | 2026-01-30 |
| Redis 메모리 이슈 해결 | 장애 대응 - Redis 메모리 이슈 해결 | 2026-01-30 |

## 업무 항목 관리

### 수정

각 업무 항목의 **제목**, **내용**, **날짜**를 클릭하여 직접 수정할 수 있습니다.

### 링크 추가

업무 항목에 관련 링크(Jira, Confluence 등)를 추가할 수 있습니다.
- 항목의 링크 아이콘을 클릭
- URL을 입력하고 저장

::: warning 안전한 URL만 허용
`http://` 또는 `https://` 프로토콜만 허용됩니다.
:::

### 삭제

업무 항목의 삭제 버튼을 클릭하면 해당 항목이 삭제됩니다.

## 공간 구조

FREE는 4단계 공간 구조를 가집니다:

```
팀 (Team)
├── 그룹 (Group)
│   ├── 파트 (Part)
│   │   ├── 개인 (Personal) ← 여기서 업무 입력
│   │   └── 개인 (Personal)
│   └── 파트 (Part)
└── 그룹 (Group)
```

| 공간 | 설명 | 내용 |
|------|------|------|
| **개인 공간** | 나의 업무 항목 | 직접 입력한 업무 목록 |
| **파트 공간** | 파트원들의 업무 모아보기 | 파트원 전체 업무 + 파트 보고서 |
| **그룹 공간** | 그룹 내 파트 보고서 모아보기 | 파트 보고서 취합 + 그룹 보고서 |
| **팀 공간** | 팀 전체 보고서 | 그룹 보고서 취합 + 팀 보고서 |

## 다른 사람 업무 보기

- **파트 공간**: 같은 파트원의 업무 항목을 볼 수 있습니다
- **이름 클릭**: 파트원의 이름을 클릭하면 개인 공간으로 이동합니다 (읽기 전용)
