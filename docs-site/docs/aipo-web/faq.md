# 자주 묻는 질문 (FAQ)

## 일반

### Q: AIPO for Web과 AIPO for Desktop의 차이점은?

**AIPO for Web**
- 웹 브라우저에서 사용
- AI 기반 자동 노트 정리 서비스
- 팀 협업 지원
- 사용자 직접 편집 불가

**AIPO for Desktop**
- 데스크톱 앱 (개발 중)
- 개인 포스팅 도구
- Coming Soon

---

### Q: 왜 직접 편집이 안 되나요?

AIPO for Web의 핵심 컨셉은 **"작성하기 귀찮을 때 쓰는 서비스"**입니다.

- 모든 내용 정리를 AI에게 위임
- 형식 맞추기, 구조화 작업 불필요
- 자연어로 수정 요청만 하면 됨

직접 편집이 필요하다면 마크다운으로 내보낸 후 다른 도구에서 편집하세요.

---

### Q: 어떤 내용을 입력해야 하나요?

아무 내용이나 입력해도 됩니다! 예시:

- 회의 중 메모한 내용
- 두서없는 아이디어
- 공유하고 싶은 지식
- 정리되지 않은 정보

AI가 알아서 구조화하고 적절한 위치에 저장합니다.

---

## 사용법

### Q: 노트가 어디에 저장되나요?

AI가 내용을 분석하여 자동으로 결정합니다:

1. 기존 폴더에 맞는 내용 → 해당 폴더에 저장
2. 새로운 주제 → 새 폴더 생성 후 저장
3. 기존 노트와 관련 → 기존 노트 업데이트

---

### Q: 원하는 폴더에 저장하고 싶어요

입력 시 힌트를 주세요:

```
"이 내용을 '2024년 회의록' 폴더에 저장해줘.
오늘 팀 회의 내용인데..."
```

---

### Q: 검색이 잘 안 돼요

검색 팁:
- 구체적인 키워드 사용
- 맥락 정보 포함 (날짜, 사람, 주제)
- 문장 형태로 검색

**나쁜 예:** "회의"
**좋은 예:** "1월에 김팀장이랑 한 마케팅 예산 회의"

---

### Q: 처리가 너무 오래 걸려요

처리 시간은 내용 복잡도에 따라 다릅니다:

| 내용 | 예상 시간 |
|------|----------|
| 간단한 메모 | 10-20초 |
| 일반 회의록 | 20-40초 |
| 긴 문서 | 40-60초 |

다른 사용자의 요청이 앞에 있으면 대기 시간이 추가됩니다.

## 협업

### Q: 팀원에게 노트를 공유하려면?

**방법 1: 팀 공간 사용**
- 팀 공간에서 작성한 노트는 자동으로 팀원에게 공유

**방법 2: 공유 링크**
- 노트 메뉴에서 "공유 링크 생성"
- 링크를 팀원에게 전달
- SSO 로그인 필요

---

### Q: 개인 노트를 팀 공간으로 옮기려면?

현재 공간 간 이동은 지원하지 않습니다.

대안:
1. 팀 공간으로 전환
2. 같은 내용으로 새 노트 작성 요청
3. 개인 공간의 원본 삭제

---

### Q: 다른 팀의 노트를 볼 수 있나요?

아니요. 팀 공간은 해당 팀원만 접근할 수 있습니다.

예외:
- Super Admin은 모든 공간 접근 가능
- 공유 링크로 접근 (SSO 로그인 필요)

---

## 트러블슈팅

### Q: 로그인이 안 돼요

확인 사항:
1. SSO 로그인 페이지 접속 가능 여부
2. 사내 네트워크 연결 상태
3. 브라우저 쿠키/캐시 삭제 후 재시도

---

### Q: "Rate Limit Exceeded" 오류가 나요

분당 요청 한도를 초과했습니다:
- 입력: 분당 5회
- 검색: 분당 10회

1분 후 다시 시도하세요.

---

### Q: 노트가 사라졌어요

확인 순서:
1. **공간 확인**: 개인/팀 공간 전환
2. **휴지통 확인**: 삭제된 노트는 30일간 보관
3. **검색**: 노트명이나 내용으로 검색

---

### Q: 번역이 안 돼요

번역 실패 시:
1. 노트 보기 화면에서 해당 언어 탭 선택
2. "번역 재시도" 버튼 클릭

계속 실패하면 관리자에게 문의하세요.
