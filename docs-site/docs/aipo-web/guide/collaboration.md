# 팀 협업

## 개인 공간 vs 팀 공간

AIPO는 로그인 시 자동으로 두 개의 공간을 생성합니다.

| 항목 | 개인 공간 | 팀 공간 |
|------|----------|---------|
| **접근 권한** | 본인만 | 같은 팀원 전체 |
| **용도** | 개인 메모, 드래프트 | 회의록, 공유 문서 |
| **노트 공유** | 불가 | 자동 공유 |

### 공간 전환 방법

사이드바 상단에서 **[개인]** / **[팀]** 탭을 클릭하여 전환합니다.

::: tip
팀에 공유할 내용은 반드시 **팀 공간을 선택한 상태**에서 입력하세요.
현재 선택된 공간에 노트가 저장됩니다.
:::

### 확인 방법

- 사이드바 상단에 현재 선택된 공간이 하이라이트됩니다
- 팀 공간에 저장한 노트가 팀원 화면에도 나타나면 정상

## 블록 댓글

노트의 특정 블록(섹션)에 댓글을 달아 팀원과 토론할 수 있습니다.

### 댓글 작성

1. 노트 보기 화면에서 원하는 블록 옆의 **말풍선 아이콘** 클릭
2. 댓글 내용 입력
3. **전송** 클릭

### 답글 달기

- 기존 댓글 하단의 **"답글"** 클릭
- 쓰레드 형태로 대화 가능

### 확인 방법

- 댓글이 달린 블록 옆에 말풍선 아이콘과 댓글 수가 표시됩니다
- 클릭하면 해당 블록의 댓글 목록을 확인할 수 있습니다

## 공유 링크

노트를 팀원에게 직접 전달하고 싶을 때 사용합니다.

### 링크 생성

1. 노트의 **메뉴 버튼**(⋯) 클릭
2. **"링크 공유"** 선택
3. 링크가 클립보드에 복사됩니다

### 전달

복사된 링크를 메신저나 이메일로 전달하세요.

::: warning 주의
공유 링크는 **SSO 로그인한 사내 사용자만** 접근할 수 있습니다.
외부 공유는 지원하지 않습니다.
:::

## 팀 관리자 기능

팀 관리자(Team Admin)로 지정되면 추가 기능을 사용할 수 있습니다.

### 일반 사용자 vs 팀 관리자

| 기능 | 일반 사용자 | 팀 관리자 |
|------|:-----------:|:---------:|
| 노트 열람 / 작성 | O | O |
| 본인 노트 삭제 | O | O |
| **모든 팀 노트 삭제** | X | O |
| **폴더 구조 Refactoring** | X | O |
| **팀 감사 로그 조회** | X | O |
| **휴지통 영구 삭제** | X | O |

### Refactoring 기능

팀 관리자는 AI에게 폴더 구조 재정리를 요청할 수 있습니다:

```
"팀 노트를 날짜별로 다시 정리해줘"
"회의록과 기획서를 분리해줘"
```

### 팀 관리자 신청

팀 관리자 지정은 Super Admin(syngha.han)에게 요청하세요.

## 감사 로그

팀 관리자는 **설정 > 감사 로그**에서 팀 활동 이력을 확인할 수 있습니다:

- 누가, 언제, 어떤 작업을 했는지 추적
- 노트 생성/수정/삭제 이력
- 댓글 활동 이력
