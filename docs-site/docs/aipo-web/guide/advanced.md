# 고급 기능

## 히스토리 (변경 이력)

노트가 수정될 때마다 이전 버전이 자동으로 저장됩니다.

- **보관 기간**: 30일
- **보관 단위**: 언어별 (한국어, 영어, 중국어 각각)

### 히스토리 보기

1. 노트 보기 화면에서 **시계 아이콘** 클릭
2. 변경 이력 목록 확인
3. 원하는 버전 선택하여 미리보기

### 확인 방법

- 시계 아이콘 클릭 시 날짜별 버전 목록이 나타나면 정상
- 각 버전을 클릭하면 해당 시점의 내용을 확인할 수 있습니다

## 마크다운 내보내기

노트를 `.md` 파일로 다운로드할 수 있습니다.

### 내보내기 방법

1. 노트의 **메뉴 버튼**(⋯) 클릭
2. **"Markdown 내보내기"** 선택
3. 옵션 선택:
   - **언어 선택**: 한국어 / 영어 / 중국어
   - **댓글 포함 여부**: 댓글도 함께 내보낼지 선택

### 확인 방법

- 브라우저의 다운로드 폴더에 `.md` 파일이 저장됩니다
- 마크다운 에디터나 텍스트 편집기에서 열어 확인할 수 있습니다

## 휴지통 관리

삭제된 노트는 바로 삭제되지 않고 **휴지통**에 30일간 보관됩니다.

### 휴지통 보기

사이드바 하단의 **휴지통** 아이콘 클릭

### 복원

1. 휴지통에서 복원할 노트 선택
2. **"복원"** 버튼 클릭
3. 원래 위치로 복원됩니다

### 영구 삭제

::: danger 주의
영구 삭제된 노트는 **복구할 수 없습니다.**
팀 관리자만 영구 삭제할 수 있습니다.
:::

### 확인 방법

- 휴지통에서 삭제된 노트 목록이 보이면 정상
- 복원 후 좌측 트리에 노트가 다시 나타나면 정상

## 번역 재시도

번역이 실패하거나 품질이 낮은 경우 다시 번역을 요청할 수 있습니다.

1. 노트 보기 화면에서 해당 언어 탭 선택
2. **"번역 재시도"** 버튼 클릭
3. AI가 다시 번역을 수행합니다

### 확인 방법

- 번역 완료 후 언어 탭의 내용이 갱신되면 정상

## 설정

우측 상단의 **설정 아이콘**에서 다음을 변경할 수 있습니다:

| 설정 | 옵션 |
|------|------|
| **인터페이스 언어** | 한국어 / 영어 / 중국어 |
| **테마** | 라이트 / 다크 / 시스템 설정 따르기 |

## 관리자 도구

### Super Admin

전체 시스템을 관리하는 최고 관리자입니다.

**사용 가능 기능:**
- 전체 팀 관리
- 모든 공간의 노트 접근/삭제
- 팀 관리자 지정/해제
- 전체 감사 로그 조회
- 시스템 통계 확인
- LLM 모델 설정

### Team Admin

Super Admin이 지정한 팀별 관리자입니다.

**사용 가능 기능:**
- 팀 내 모든 노트 삭제
- 팀 폴더 구조 Refactoring
- 팀 감사 로그 조회
- 팀 휴지통 영구 삭제

### 확인 방법

- 관리자 권한이 있으면 설정 메뉴에 **"관리"** 탭이 추가로 표시됩니다
- 팀 관리자는 팀 노트에 대한 삭제 버튼이 모든 노트에 표시됩니다
