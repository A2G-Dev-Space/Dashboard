# Jarvis 장기기억 & 모델 관리

Knox Messenger 채팅을 통해 Jarvis의 장기기억과 LLM 모델을 자연어로 관리할 수 있습니다.

## 장기기억 (Long-term Memory)

Jarvis는 대화 중 중요한 정보를 영구적으로 기억합니다.
앱을 재시작해도 유지됩니다.

### 기억 확인

- "장기기억 전부 알려줘"
- "뭘 기억하고 있어?"
- "내가 저번에 뭐라고 했었지?"

### 기억 추가

- "이거 기억해: 매주 금요일 주간보고 마감"
- "내 선호 모델은 GLM5야"

### 기억 삭제

- "주간보고 마감 기억 지워줘"
- "장기기억 중에 오래된거 정리해줘"

### 자동 저장

대화 컨텍스트가 모델 한계의 70%를 초과하면 Jarvis가 자동으로 중요한 내용을 장기기억에 저장한 후 대화를 압축합니다.

저장 시 Knox Messenger에 "💾 장기기억 자동 저장" 알림이 표시됩니다.

## 모델 관리

### 현재 모델 확인

- "넌 모델이 뭐야?"
- "지금 뭘로 돌고 있어?"

### 사용 가능한 모델 목록

- "모델 뭐 쓸 수 있어?"
- "다른 모델 있어?"

### 모델 변경

- "3번으로 바꿔줘"
- "GLM5로 바꿔줘"
- "가장 빠른 모델로 바꿔줘"

모델 변경 시 기존 대화 히스토리는 유지됩니다.
Knox Messenger에 "🔄 모델 변경" 알림이 표시됩니다.

## 주의사항

- 장기기억은 최대 100개까지 저장됩니다 (초과 시 오래된 것부터 자동 삭제).
- 모델 변경은 Jarvis 세션 동안만 유지됩니다. 재연결 시 기본 모델로 돌아갑니다.
- 본 Agent의 응답 품질과 속도는 사용 중인 LLM 모델의 성능에 크게 좌우됩니다.
