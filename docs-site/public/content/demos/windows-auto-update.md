# Windows 자동 업데이트

Nexus Bot Desktop 앱의 자동 업데이트 과정을 확인하세요.

## 작동 방식

1. **자동 확인** — 앱 시작 시 최신 버전을 자동으로 확인합니다
2. **알림 표시** — 새 버전이 있으면 업데이트 알림이 표시됩니다
3. **백그라운드 다운로드** — 승인하면 백그라운드에서 다운로드합니다
4. **자동 설치** — 다운로드 완료 후 앱 재시작 시 자동 설치됩니다

## 팁

- 네트워크 연결이 필요합니다
- 업데이트 중에도 작업을 계속할 수 있습니다
- 설치 후 앱이 자동으로 재시작됩니다
