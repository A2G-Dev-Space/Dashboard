# Knox Messenger 연동 (Jarvis)

Knox Messenger를 통해 어디서든 내 PC의 AI 비서를 원격으로 조작할 수 있습니다.

## 시작하기

### 1. Nexus Bot 실행

PC에서 Nexus Bot (For Windows)을 실행합니다.

### 2. 자비스 연결

하단 도구 모음에서 **로봇 아이콘 (자비스 연결)** 버튼을 클릭합니다.

- 🔘 회색 = 미연결
- 🟢 초록색 = 연결됨
- 🟡 노란색 = 연결 중...

### 3. Knox Messenger에서 대화

연결 성공 시 Knox Messenger에 Jarvis가 인사 메시지를 보냅니다.
이후 메시지를 보내면 Jarvis가 응답합니다.

## 주요 기능

### 원격 작업 지시

Knox Messenger에서 자연어로 작업을 지시하면 PC의 Nexus Bot이 실행합니다.

- "Jira에서 나한테 할당된 이슈 정리해줘"
- "어제 커밋한 내용 요약해줘"
- "LLM 성능 비교 PPT 만들어줘"

### 모델 관리

- "넌 모델이 뭐야?" — 현재 사용 중인 모델 확인
- "모델 뭐 쓸 수 있어?" — 사용 가능한 모델 목록
- "3번으로 바꿔줘" — 모델 변경

### 장기기억 관리

- "장기기억 전부 알려줘" — 저장된 기억 목록
- "이거 기억해줘: ..." — 새 기억 추가
- "그 기억 지워줘" — 기억 삭제

### 자동 체크 (Polling)

근무시간(KST 9~18시) 동안 입력이 없으면 자동으로 확인합니다.

- Jira 할당 이슈
- Confluence 멘션 페이지
- FREE TODO 목록

Settings에서 체크 주기와 시간대를 변경할 수 있습니다.

## 연결 해제

자비스 연결 버튼을 다시 클릭하면 연결이 해제됩니다.
앱을 재시작하면 연결이 자동으로 초기화됩니다 (수동 연결 필요).

## 주의사항

- 본 Agent의 응답 품질과 속도는 사용 중인 LLM 모델의 성능에 크게 좌우됩니다.
- Knox Messenger를 통한 응답은 보안을 위해 제목 수준의 요약만 전달됩니다.
- PC가 꺼져있거나 Nexus Bot이 종료된 상태에서는 사용할 수 없습니다.
